Lino UI Developer
=================

**Requirements**

- You are familiar with Python_ and Django_ 
- You are also familiar with ExtJS_ or some other Javascript framework
- You have seen the `Lino online demos
  <http://lino-framework.org/demos>`_ and noticed that they are still
  using ExtJS 3.
- You have worked your way through the *Getting started* section of
  the `Lino Developer's Guide <http://lino-framework.org/dev>`_ and
  got your local Lino project up and running.
- You have forked the Lino code repository on GitHub and submitted
  some pull requests.
- Your job will be to write a new package, similar to the existing
  :mod:`lino.modlib.extjs` package, which uses ExtJS 5 or some other
  Javascript framework.
